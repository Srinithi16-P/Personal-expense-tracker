# SmartSpend – Personal Expense Tracker

A modern **Personal Expense Tracker** web application that helps users efficiently manage their personal finances by tracking income, recording expenses, creating budgets, setting financial goals, and analyzing spending patterns through interactive dashboards and reports.

---

# Project Abstract

Managing personal finances is an essential part of everyday life, yet many individuals struggle to monitor their income, expenses, and savings effectively.

**SmartSpend** is a full-stack web application designed to simplify financial management by providing a secure and centralized platform where users can organize all their financial activities in one place.

The application enables users to:

- Record income and expenses
- Categorize financial transactions
- Create monthly budgets
- Set savings goals
- Analyze spending habits
- Generate downloadable reports
- Receive budget and goal notifications
- Secure user accounts using JWT Authentication

Through interactive dashboards, financial analytics, and intelligent budgeting tools, SmartSpend helps users make informed financial decisions while encouraging disciplined saving and spending habits.
